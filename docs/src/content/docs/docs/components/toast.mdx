### Position

You can change where all toasts are displayed via the `position` prop on the `<Toaster />` component:

```astro /position='[a-z-]+'/ showLinenumbers title="ToastPosition.astro"
---
import { Toaster } from 'studiocms:ui/components';
---

<Toaster position='top-left' />
<Toaster position='top-center' />
<Toaster position='top-right' />
<Toaster position='bottom-left' />
<Toaster position='bottom-center' />
<Toaster position='bottom-right' />
```

### Duration

There's two ways to set a toast's duration: via the default on the `<Toaster />` component or via the `toast({ ... })` function.

You can set the default duration like this:

```astro "duration={10000}" showLinenumbers title="ToastDuration.astro"
---
import { Toaster } from 'studiocms:ui/components';
---

<Toaster duration={10000} />
```

Or you can set it individually for every toast like this:

```ts "duration: 10000" twoslash showLinenumbers title="ToastDuration.ts"
// @noErrors
import { toast } from 'studiocms:ui/components';

toast({
  title: 'This is a toast!',
  description: 'Hi there!',
  type: 'info',
  duration: 10000
});
```

### Persistence

Sometimes, you don't need a toast to expire after a set time. Instead of setting the toast to only expire after 31 days, you can set it
to be persistent by setting `persistent` to `true`:

```ts "persistent: true" twoslash showLinenumbers title="ToastPersistence.ts"
// @noErrors
import { toast } from 'studiocms:ui/components';

toast({
  title: 'This is a toast!',
  description: 'Hi there!',
  type: 'info',
  persistent: true
});
```

### Close Button

Unless a toast is persistent, you can choose to hide the close button by setting the `closeButton` option to `false`:

```ts "closeButton: false" twoslash showLinenumbers title="ToastCloseButton.ts"
// @noErrors
import { toast } from 'studiocms:ui/components';

toast({
  title: 'This is a toast!',
  description: 'Hi there!',
  type: 'info',
  closeButton: false
});
```

### Offset & Gap

You can change how far the toasts are away from the screen's edge by changing the `offset` prop on the `<Toaster />`:

```astro showLinenumbers title="ToastOffset.astro"
---
import { Toaster } from 'studiocms:ui/components';
---

<!-- Offset in pixels -->
<Toaster offset={64} />
```

You can also change by how much toasts are separated via the `gap` prop on the `<Toaster />`:

```astro showLinenumbers title="ToastGap.astro"
---
import { Toaster } from 'studiocms:ui/components';
---

<!-- Gap in pixels -->
<Toaster gap={32} />
```
