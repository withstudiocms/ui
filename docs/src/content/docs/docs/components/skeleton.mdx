An extremely customizable skeleton loading component.

## Usage

<Tabs>
  <TabItem label='Preview'>
    <PreviewCard>
      <Skeleton variant='card'>
        <Skeleton variant='block' width='50%' height='8rem' />
        <Skeleton variant='text' />
        <Skeleton variant='text' />
      </Skeleton>
    </PreviewCard>
  </TabItem>
  <TabItem label='Code'>
    ```astro showLinenumbers title='SkeletonExample.astro'
    ---
    import { Skeleton } from 'studiocms:ui/components';
    ---

    <Skeleton variant='card'>
      <Skeleton variant='block' width='50%' height='8rem' />
      <Skeleton variant='text' />
      <Skeleton variant='text' />
    </Skeleton>
    ```
  </TabItem>
</Tabs>

### Variant

You can use the `variant` prop to change the skeleton's appearance. The following variants are available:

- `card`
- `text`
- `block`
- `circle`

<Tabs>
  <TabItem label='Preview'>
    <PreviewCard>

          <Skeleton variant='card'></Skeleton>
          <Skeleton variant='text'></Skeleton>
          <Skeleton variant='block' width='50%' height='8rem'></Skeleton>
          <Skeleton variant='circle' width='100px' height='100px'></Skeleton>
        </div>
    </PreviewCard>
  </TabItem>
  <TabItem label='Code'>
    ```astro "variant='card'" "variant='text'" "variant='block'" "variant='circle'" showLinenumbers title='SkeletonExample.astro'
    ---

    ---

    <div style='width: 100%; display: flex; flex-direction: column; gap: 1rem;'>

      <Skeleton variant='block' width='50%' height='8rem'></Skeleton>
      <Skeleton variant='circle' width='100px' height='100px'></Skeleton>
    </div>
    ```
  </TabItem>

The `width` and `height` props can be used to set the width and height of the skeleton. The value can be a string ending in `%` or `px` or `rem` to specify the size in percentage, pixels or rem respectively.

## Card variant

The `Skeleton` component is meant to be used as a container for other `Skeleton` components.
There is no animation state for the `card` variant. By default, the `card` variant will take up the full width of its container.

The `Skeleton` component also supports the following props for the `card` variant:

- `direction`
- `hAlign`
- `vAlign`
- `gap`

### Direction

You can use the `direction` prop to change the direction of the skeleton. The following directions are available:

- `row`
- `column`

The default value is `column`.
