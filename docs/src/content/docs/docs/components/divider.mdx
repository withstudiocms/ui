## Changing the label background color

There may be cases where your divider does not sit on an element with the default page background color. In such cases,
you can use the `background` prop to change it to match the correct color.

```astro "background='background-step-1'" showLinenumbers title="DividerBackgroundExample.astro"
---
import { Divider } from '@studiocms/ui/components';
---

<Divider background='background-step-1'>
  Divider
</Divider>
```
