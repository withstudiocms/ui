A simple card component to wrap other stuff in! 

## Usage

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Card>
        Content
      </Card>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro showLinenumbers title="CardExample.astro"
    ---
    import { Card } from 'studiocms:ui/components';
    ---

    <Card>
      Content
    </Card>
    ```
  </TabItem>
</Tabs>

### Variants <Badge text="New!" variant="tip" />

Instead of just having a bordered card, you can use the `variant` prop to change the card's appearance. Set it to
`filled` to have a card with a filled background:

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Card variant="filled">
        Content
      </Card>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro "variant='filled'" showLinenumbers title="FullVariantExample.astro"
    ---
    import { Card } from 'studiocms:ui/components';
    ---

    <Card variant='filled'>
      Content
    </Card>
    ```
  </TabItem>
</Tabs>

### Full Width & Full Height

You can set the card to grow to its parent containers width using the `fullWidth` param:

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Card fullWidth>
        Content
      </Card>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro "fullWidth" showLinenumbers title="FullVariantExample.astro"

    ---

      Content
    </Card>
    ```

  </TabItem>
</Tabs>

### Header & Footer <Badge text="New!" variant="tip" />

The card component supports a header and footer slot:

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Card>
        <h4 slot='header'>Header Content</h4>
        <p>
          Lorem ipsum dolor sit amet. Et nulla cumque et suscipit molestiae est quam eaque et placeat iste est 
          velit dignissimos aut totam voluptatum qui eveniet nostrum. Aut neque reprehenderit est vero debitis 
          in inventore voluptas ut dolores facilis et voluptatibus quibusdam aut dolorum sunt non vitae sequi.
        </p>
        <span slot='footer'>Footer Content</span>
      </Card>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro /slot='(?:header|footer)'/ showLinenumbers title="CardSlotsExample.astro"
    ---
    import { Card } from 'studiocms:ui/components';
    ---

    <Card>
      <h4 slot='header'>Header Content</h4>
      <p>
        Lorem ipsum dolor sit amet. Et nulla cumque et suscipit molestiae est quam eaque et placeat iste est 
        velit dignissimos aut totam voluptatum qui eveniet nostrum. Aut neque reprehenderit est vero debitis 
        in inventore voluptas ut dolores facilis et voluptatibus quibusdam aut dolorum sunt non vitae sequi.
      </p>
      <span slot='footer'>Footer Content</span>
    </Card>
    ```
  </TabItem>
</Tabs>

### Polymorphic

A card can be anything you want it to be, literally! Cards support Astro's `Polymorphic` type, meaning you can render the card
as whatever you want by passing the `as` attribute. Here's a card that renders as an `<a>` tag instead of a `<div>`: 

<Tabs>
  <TabItem label="Preview">
    <PreviewCard>
      <Card as="a" href="https://studiocms.dev">
        studiocms.dev
      </Card>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro "as='a'" showLinenumbers title="CardPolymorphicExample.astro"
    ---
    import { Card } from 'studiocms:ui/components';
    ---

    <Card as='a' href='https://studiocms.dev'>
      studiocms.dev
    </Card>
    ```
  </TabItem>
</Tabs>
