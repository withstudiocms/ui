### Variants

For those who don't like the appearance of the tabs in general, you can change the `variant` prop to `starlight`. This
will give the tabs the appearance of the tabs that Starlight uses, the same look you can see for the preview cards on
this page.

<Tabs>
  <TabItem label="Preview">
    <PreviewCard fullWidth>
      <SuiTabs variant="starlight">
        <SuiTabItem label="Tab #1">
          <Card fullWidth>
            Content #1
          </Card>
        </SuiTabItem>
        <SuiTabItem label="Tab #2">
          <Card fullWidth>
            Content #2
          </Card>
        </SuiTabItem>
        <SuiTabItem label="Tab #3">
          <Card fullWidth>
            Content #3
          </Card>
        </SuiTabItem>
      </SuiTabs>
    </PreviewCard>

### Alignment

In certain cases, you might want to change the alignment of the tabs. You can do this by setting the `align` prop to 
`left`, `center` or `right`, with left being the default. Here's an example of the tabs being aligned to the right:

<Tabs>
  <TabItem label="Preview">
    <PreviewCard fullWidth>
      <SuiTabs align="right">
        <SuiTabItem label="Tab #1">
          <Card fullWidth>
            Content #1
          </Card>
        </SuiTabItem>
        <SuiTabItem label="Tab #2">
          <Card fullWidth>
            Content #2
          </Card>
        </SuiTabItem>
      </SuiTabs>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro "align='right'" showLinenumbers title="TabsAlignmentExample.astro"
    ---
    import { Card } from 'studiocms:ui/components/card';
    import { Tabs, TabItem } from 'studiocms:ui/components/tabs';
    ---

    <Tabs align='right'>
      <TabItem label='Tab #1'>
        <Card fullWidth>
          Content #1
        </Card>
      </TabItem>
      <TabItem label='Tab #2'>
        <Card fullWidth>
          Content #2
        </Card>
      </TabItem>
    </Tabs>
    ```
  </TabItem>
</Tabs>

This even works with the `starlight` variant!

### Syncing & Storage

To keep a users selection saved and even sync multiple different tab containers with each other, you can set a `syncKey`.
By default, this value will be stored in sessionStorage to comply with GDPR guidelines, however if you want to, you can
set the `storage` prop to `persistent` to store the value in localStorage.

<Tabs>
  <TabItem label="Preview">
    <PreviewCard fullWidth vertical>
      <SuiTabs syncKey="sync-example">
        <SuiTabItem label="Tab #1 (Sync 1)" icon="heroicons:bolt">
          <Card fullWidth>
            Tab Contents #1 (Sync 1)
          </Card>
        </SuiTabItem>
        <SuiTabItem label="Tab #2 (Sync 2)" icon="heroicons:bolt-slash" color="success">
          <Card fullWidth>
            Tab Contents #2 (Sync 2)
          </Card>
        </SuiTabItem>
      </SuiTabs>
      <Divider>Different Element</Divider>
      <SuiTabs syncKey="sync-example">
        <SuiTabItem label="Tab #3 (Sync 1)" icon="heroicons:bolt">
          <Card fullWidth>
            Tab Contents #3 (Sync 1)
          </Card>
        </SuiTabItem>
        <SuiTabItem label="Tab #4 (Sync 2)" icon="heroicons:bolt-slash" color="success">
          <Card fullWidth>
            Tab Contents #4 (Sync 2)
          </Card>
        </SuiTabItem>
      </SuiTabs>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">

### Nesting Tabs

You can nest tabs inside of other tabs, allowing for complex tab structures. Here's an example of how to do this:

<Tabs>
  <TabItem label="Preview">
    <PreviewCard fullWidth>
      <SuiTabs>
        <SuiTabItem label="Tab #1 (Normal)">
          <Card fullWidth>
            Tab Contents #1 (Normal)
          </Card>
        </SuiTabItem>
        <SuiTabItem label="Tab #2 (Nested)">
          <Card fullWidth>
            <SuiTabs>
              <SuiTabItem label="Nested Tab #1">
                <Card fullWidth>
                  Nested Tab Contents #1
                </Card>
              </SuiTabItem>
              <SuiTabItem label="Nested Tab #2">
                <Card fullWidth>
                  Nested Tab Contents #2
                </Card>
              </SuiTabItem>
            </SuiTabs>
          </Card>
        </SuiTabItem>
      </SuiTabs>
    </PreviewCard>
  </TabItem>
  <TabItem label="Code">
    ```astro "syncKey='sync-example'" showLinenumbers title="TabsSyncExample.astro"
    ---
    import { Divider } from 'studiocms:ui/components/divider';

    import { Tabs, TabItem } from 'studiocms:ui/components/tabs';

    <Tabs>

    ```
  </TabItem>
</Tabs>
