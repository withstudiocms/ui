组组件是一个容器组件，用于将其子元素分组在一起。
它非常适合对按钮、徽章、输入框和其他组件进行分组，
从而更轻松地管理它们的布局。它还会改变它们的样式，使它们看起来像是一个条形。

组组件支持以下组件：

- Button（按钮）
- Badge（徽章）
